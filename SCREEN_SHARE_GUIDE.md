# 屏幕共享选择指南

## 功能概述

你的项目现在支持多种屏幕共享选项，用户可以根据需要选择不同的共享方式。

## 使用方法

### 1. 基本屏幕共享

点击"共享屏幕"按钮，浏览器会弹出选择对话框，让你选择：

- **整个屏幕** - 共享整个显示器
- **应用程序窗口** - 共享特定应用程序窗口
- **浏览器标签页** - 共享当前浏览器标签页

### 2. 增强的屏幕共享选项

现在界面提供了下拉菜单，包含以下选项：

#### 共享整个屏幕

- 共享整个显示器
- 适合演示整个桌面环境
- 包含所有打开的应用程序

#### 共享应用程序窗口

- 只共享特定的应用程序窗口
- 适合演示特定软件
- 更精确的内容控制

#### 共享浏览器标签页

- 只共享当前浏览器标签页
- 适合网页演示
- 最轻量的共享方式

## 技术实现

### 代码位置

- 主要逻辑：`src/hooks/liveVideo/useShareScreen.tsx`
- 界面组件：`src/components/LiveVideo/LiveFooter/LiveFooter.tsx`

### 关键参数

```typescript
const constraints = {
  video: {
    displaySurface: 'monitor' | 'window' | 'browser',
    cursor: 'always',
    width: { ideal: 1920 },
    height: { ideal: 1080 },
  },
  audio: {
    echoCancellation: true,
    noiseSuppression: true,
    sampleRate: 44100,
  },
};
```

## 注意事项

1. **权限要求**：首次使用需要用户授权屏幕共享权限
2. **性能考虑**：共享整个屏幕比共享窗口消耗更多资源
3. **隐私保护**：共享时会显示红色边框提示用户正在共享
4. **音频共享**：可以选择是否同时共享系统音频

## 故障排除

### 常见问题

1. **无法选择屏幕**：检查浏览器权限设置
2. **共享质量差**：尝试降低分辨率或关闭其他应用程序
3. **音频问题**：检查系统音频设置和浏览器权限

### 调试方法

打开浏览器开发者工具，查看控制台错误信息，或检查网络连接状态。
